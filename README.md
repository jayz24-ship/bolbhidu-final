# BolBhidu - AI Powered Civic Issue Reporting Platform

BolBhidu is an AI-powered civic issue reporting and community engagement platform that enables citizens to report, verify, and track local civic problems in real time. The platform combines AI-based content verification, geolocation-based feeds, and live community interaction features to improve transparency and civic participation.

---

## Features

### User Features
- Report civic issues with images, descriptions, and location data
- Location-based community issue feeds
- Real-time likes, comments, shares, and notifications
- Live issue progress tracking
- JWT-based secure authentication and authorization
- Responsive modern UI using Tailwind CSS

### AI-Powered Verification
- Automated spam detection using OpenAI APIs
- AI-based authenticity verification for uploaded content
- Intelligent moderation support for suspicious posts

### Admin Features
- Admin dashboard for issue management
- SLA-based issue tracking and workflow handling
- Content moderation system
- Strike-based banning system for abusive users
- Progress status updates for reported issues

### Backend & System Features
- Scalable REST API architecture
- Real-time communication using Socket.io
- Geospatial queries for nearby issue discovery
- Cloud media upload and handling
- MongoDB database integration
